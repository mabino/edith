Edith

A native macOS text editor built with Swift and SwiftUI

macOS 13.0+ Swift 5.9 MIT License
View on GitHub Download

📝 Multi-Document

Work with multiple files in a tabbed interface with session restore.

🔢 Line Numbers

Click or drag line numbers to select entire lines. Cmd+click for non-contiguous.

🎨 Syntax Highlighting

Support for 50+ programming languages via HighlightSwift.

🔍 Find & Replace

Full regex/PCRE support with document selector and Extract All.

👁️ Invisible Characters

See spaces, tabs, and line endings with custom character rendering.

🔄 File Monitoring

Automatic detection when files change on disk with reload option.

🟢 Vim-like Mode (Experimental)

Enable in Settings → General. Double-tap Esc to toggle. Green glow indicates normal mode.

h j k l navigate
w b e by word
gg G start/end
i a o insert
x dd delete
